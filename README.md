# Contact Page

A personal contact page for Robinson Wesonga, a website developer and graphic designer.

## Description

This project is a responsive contact page featuring a contact form and links to various social media platforms and contact methods. The design includes modern CSS styling with gradients, animations, and a dark theme.

## Features

- **Contact Form**: Includes fields for name, email, phone number, service type, message, and password
- **Social Media Links**: Direct links to Facebook, Twitter, Instagram, LinkedIn, GitHub, YouTube, Pinterest, TikTok, and WhatsApp
- **Contact Information**: Phone number and email address
- **Responsive Design**: Adapts to different screen sizes with mobile-friendly layout
- **Modern Styling**: Uses CSS gradients, backdrop filters, and hover effects for a polished look

## Technologies Used

- HTML5
- CSS3
- Font Awesome (for icons)

## How to View

1. Clone or download the project files
2. Open `contact.html` in any modern web browser
3. The page will load with all styling and functionality
